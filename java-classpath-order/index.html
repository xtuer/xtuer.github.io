  • Java
  • SpringBoot

提示:

  • 使用 *，不能使用 *.jar
  • 使用 lib/* 只能查找 lib 目录下的 jar，不能递归查找 lib 子目录下的 jar
  • 使用 lib/* 时，哪个 jar 排在前面由系统对文件排序决定
