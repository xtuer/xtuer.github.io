Tomcat 多域名

  • Java

一个 Tomcat 可以支持多个域名，需要在 server.xml 中为每一个域名配置一个 Host，参考默认的 localhost 的配置。

默认 Host

defaultHost 是指使用 IP 访问时 Tomcat 默认用来处理请求的 Host，也就是访问此域名对应的应用。如果有多个域名时，最好是配置一下。
