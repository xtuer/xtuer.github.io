Nginx + Https

一般不会在 Spring Boot 应用上直接启用 Https，而是 Spring Boot 继续提供 Http 服务，Nginx 把 Https 请求作为 Http 请求转发到 Spring Boot，Nginx 部分的配置可参考 Nginx + Tomcat 使用 Https，也可参考 SpringBoot + Nginx + Https 下 Redirect 问题，Spring Boot 部分需要在 application.properties 增加下面的配置:

server.tomcat.remote_ip_header = x-forwarded-for
server.tomcat.protocol_header  = x-forwarded-proto
server.tomcat.port-header      = X-Forwarded-Port
server.use-forward-headers     = true
