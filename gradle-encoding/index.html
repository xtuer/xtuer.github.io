  • Gradle

Gradle 默认使用系统字符编码 (Windows 为 GBK，Linux 和 Mac 为 UTF-8)，很多程序员都是使用 Windows，但是 Java 文件以及其他资源文件大多数都会使用 UTF-8 (因为要跨平台使用)，在 Windows 开发时编译运行容易出现乱码，报错等。
